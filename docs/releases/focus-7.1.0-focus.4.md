# BliPai Focus v7.1.0-focus.4 发布说明

BliPai Focus `v7.1.0-focus.4` 基于上游 `BiliPai v7.1.0`，这一版主要收口 Focus 的项目定位、文档展示和维护说明，同时继续保持当前 Focus 功能链路不变。

## ✨ 重点更新

- 重写 Focus 简介，明确项目核心思想：尽量去掉一切会把人拉进被动刷取的推荐入口，只保留更偏主动搜索、主动订阅和主动回看的使用路径。
- 从注意力心理学和行为设计角度补充了 Focus 的解释文案，强调“少诱饵、保留目的”的产品取向。
- README / README_EN 现已同步成统一的 Focus 图库结构，并新增“历史记录一键清空”截图展示。
- “应用预览”已明确标注为“官方应用预览”，并说明后续大部分说明内容主要沿用上游官方原始 README。
- README 现已补充 Focus 主要 diff 文件、测试设备范围，以及“尽量与主项目保持同主版本号，暂定只跟进大版本”的维护节奏说明。
- Star History 已切换为 Focus 仓库自身的数据。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
