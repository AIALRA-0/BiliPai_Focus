# BliPai Focus v7.1.0-focus.5 发布说明

BliPai Focus `v7.1.0-focus.5` 基于上游 `BiliPai v7.1.0`，这一版主要修复关注分组管理在 0 对象场景下的体验问题，并补齐大关注列表下更高效的分组搜索能力。

## ✨ 重点更新

- 修复动态页侧栏、动态页横向关注列表和首页“关注”分类在没有可用关注对象时的空态抖动问题，统一改成稳定显示“没有可用关注对象”。
- “分组管理”新增搜索框，放在新分组输入区下方，可按 UP 名称或 UID 快速筛出目标对象。
- 统一新分组输入框与“添加”按钮的高度、圆角和居中对齐，输入区视觉更整洁。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `focus.5` 版本线与发布入口。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
