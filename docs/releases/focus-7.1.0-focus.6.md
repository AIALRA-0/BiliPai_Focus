# BliPai Focus v7.1.0-focus.6 发布说明

BliPai Focus `v7.1.0-focus.6` 基于上游 `BiliPai v7.1.0`，这一版主要继续收口关注分组管理面板的输入区样式，并修复动态页与首页“关注”在 Focus 过滤后的补页滞后和空态连闪问题。

## ✨ 重点更新

- 继续抬高“新分组名称”和“搜索关注对象”输入框，并把 `添加 / 刷新` 按钮统一到相同高度、圆角和按钮样式，分组管理输入区更整齐。
- 修复动态页与首页“关注”在 Focus 过滤后的刷新补页策略；现在会优先补到更稳定的可见数量后再展示，减少刷新后从 1 条逐步变成 3 条的明显跳变。
- 修复过滤后完全没有可用关注对象时，动态页和首页“关注”尾部仍误触发自动加载的问题；空态会稳定停留为“没有可用关注对象”。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `focus.6` 版本线与发布入口。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
