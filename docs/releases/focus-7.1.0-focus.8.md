# BliPai Focus v7.1.0-focus.8 发布说明

BliPai Focus `v7.1.0-focus.8` 基于上游 `BiliPai v7.1.0`，这一版主要继续优化首页“关注”的首屏补页体验，并把动态页关注对象的默认同步预算从 `50` 位提升到最多 `1000` 位。

## ✨ 重点更新

- 首页“关注”在过滤后如果可见视频还没达到更稳定的首屏数量，会继续主动补拉更多动态页，直到补到目标数量级或确实没有更多数据，避免必须连续刷新多次才逐渐显示完全。
- 动态页关注对象默认同步预算提升到最多 `1000` 位；如果实际关注数不足 `1000`，会直接同步实际总数，不再默认只停留在第一页的 `50` 位。
- 旧版本遗留下来的小关注缓存也会在当前版本里自动补拉，不必额外等缓存 TTL 过期。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `focus.8` 版本线与发布入口。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
