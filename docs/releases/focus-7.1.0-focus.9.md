# BliPai Focus v7.1.0-focus.9 发布说明

BliPai Focus `v7.1.0-focus.9` 基于上游 `BiliPai v7.1.0`，这一版主要继续优化首页“关注”的刷新体感，把首屏展示改成“先更快看到第一批视频，再后台继续补齐”。

## ✨ 重点更新

- 首页“关注”刷新时不再为了尽量一次性补满结果而长时间阻塞首屏；现在会先以前台少量补页快速给出第一批可见视频。
- 首屏内容展示出来后，如果仍有必要补齐，会在后台继续拉取更多动态页，把关注流补到更稳定的数量级。
- 保留上一版的关注对象默认同步 `1000` 位策略，不会退回到只同步 `50` 位。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `focus.9` 版本线与发布入口。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
