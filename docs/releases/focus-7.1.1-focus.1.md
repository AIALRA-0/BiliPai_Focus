# BliPai Focus v7.1.1-focus.1 发布说明

BliPai Focus `v7.1.1-focus.1` 基于上游 `BiliPai v7.1.1`。这一版的重点是合并上游语言与沉浸式修复，并把首页“关注”过滤刷新链路重新建立在官方 `HOME_FOLLOW` 分页之上。

## ✨ 重点更新

- 合并上游 `v7.1.1` 的应用语言切换、首页多语言文案、深色风格拆分与沉浸式安全区修复。
- 首页“关注”改成“官方 `HOME_FOLLOW` 分页 + Focus 过滤后补足本轮可见增量”的两阶段策略，不再经常误报“没有可用关注对象”，也不再只剩 `1-2` 条结果反复跳变。
- Focus 关注过滤关闭时，首页“关注”继续保持上游官方刷新行为，避免无过滤场景继续偏离官方节奏。
- 顶栏安全区重新回到上游 `7.1.1` 的沉浸式基线，减少某些系统上顶栏与屏幕顶部之间出现细缝的问题。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `7.1.1-focus.1` 发布线。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
