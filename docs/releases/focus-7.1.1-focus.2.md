# BliPai Focus v7.1.1-focus.2 发布说明

BliPai Focus `v7.1.1-focus.2` 基于上游 `BiliPai v7.1.1`。这一版是一个聚焦首页“关注”稳定性的热修复版本，主要处理 Focus 分组过滤开启后出现的恶性空态和分页覆盖问题。

## ✨ 重点更新

- 修复首页“关注”在开启 Focus 分组过滤、且关闭增量刷新时，后续补页会覆盖前面已拿到视频的问题；当前轮次现在会先累加，再按 `loadMore / 增量刷新 / 非增量刷新` 投影成最终结果。
- 修复首页“关注”在 Focus 分组过滤开启后出现的恶性不稳定表现，不再容易出现“第一次 1 条、再刷新 8 条、再刷新换一批、重进后直接空”的结果飘移。
- 修复冷启动和设置流重投影时首页“关注”被过早写入“没有可用关注对象”的问题；现在只有在真正成功完成过一次关注流解析后，才允许显示这个空态。
- 保持官方 `HOME_FOLLOW` 基础分页接口与刷新语义不变，只修结果层的累加和空态时机，不额外偏离上游行为。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `7.1.1-focus.2` 发布线。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
