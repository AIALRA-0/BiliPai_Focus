# BliPai Focus v7.1.1-focus.3 发布说明

BliPai Focus `v7.1.1-focus.3` 基于上游 `BiliPai v7.1.1`。这一版重点把首页“关注”的 Focus 过滤链路收回到更接近官方的工作流，修正过滤切换后残留旧结果、刷新顺序怪异和到底部后不继续加载的问题。

## ✨ 重点更新

- 首页“关注”重新贴回官方 `HOME_FOLLOW` 的单通道分页与原始顺序，不再继续沿用 Focus 版混合补齐 / 自定义排序逻辑。
- 修复过滤开关或分组配置变化后，首页“关注”仍沿用上一轮过滤结果继续显示的问题；现在会先清空旧缓存并真正重载关注页。
- 修复首页“关注”刷新时表现成“往上塞内容”的交互错位，当前重新跟随官方分页投影方式，只在结果层做 Focus 过滤。
- 保留底部 `load more` 触发修复：当列表尺寸变化后，滚到底部会继续自然加载，不再必须切页再回来。
- README / README_EN / CHANGELOG / Focus Changelog 已同步切到 `7.1.1-focus.3` 发布线。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
