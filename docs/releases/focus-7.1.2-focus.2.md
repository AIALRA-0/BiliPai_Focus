# BliPai Focus v7.1.2-focus.2 发布说明

BliPai Focus `v7.1.2-focus.2` 继续基于上游 `BiliPai v7.1.2`。这一版不再扩展首页 title 控制范围，而是把 Focus 首页开关收回到更贴近官方的 5 项表面，并继续修复首页“关注”过滤后的刷新延迟与底部续加载停滞问题。

## ✨ 重点更新

- Focus 设置页移除了 `显示追番 / 显示知识 / 显示科技` 三个按钮；这三项重新回到原有顶部标签管理，不再由 Focus 单独控制。
- 首页“关注”过滤刷新提速：只要当前轮次已经出现首批可见视频，就立即结束前台补抓，不再为追平首个官方 chunk 而额外空等。
- 首页“关注”刷新不再同步阻塞用户信息请求，过滤补抓循环中的额外 `100ms` 人工延迟也已经移除。
- 首页“关注”拉到底后会继续尝试向后补抓，直到至少出现一条新的可见视频，或确认所有可见 UP 视频都已经真正耗尽，不再轻易卡在底部要求切页重进。
- 当 Focus 可控制的首页入口全部关闭时，当前会安全回退到单 `推荐`，避免首页因空 title 场景直接崩溃。

## 🧩 Focus 默认特性

- 首页 `推荐 / 热门 / 直播 / 游戏 / 分区` 默认隐藏
- 首页 `关注` 默认显示，可在 Focus 设置中关闭
- 首页 `追番 / 知识 / 科技` 跟随原有顶部标签管理，不再提供 Focus 单独开关
- 搜索页热门搜索默认关闭，但保留搜索建议、搜索发现、搜索结果与搜索历史
- 观看历史支持“一键清空全部历史记录”
- 视频详情页相关推荐默认隐藏，可在 Focus 设置中恢复
- 动态页与首页“关注”支持统一的本地关注分组过滤，并提供总开关
