# BliPai Focus v7.1.2-focus.3

发布日期：2026-03-24

## 更新摘要
- 首次进入的新手引导页补回 `Telegram 群` 入口，同时保留 `Focus GitHub` 快捷入口。
- Focus 安装包改用独立 `applicationId`：`com.android.purebilibili.focus`，现在可以与官方原版并行安装，不再互相覆盖。
- Focus 版更新声明进一步明确：应用内检查更新只跟随 Focus 分支，不接收官方原版更新推送。

## 首页“关注”修复
- 下拉刷新时不再把已缓存并正在显示的视频整段清空；新结果会与旧列表前插合并，旧内容继续保留。
- Focus 过滤开启时，首页“关注”会继续补抓，直到本轮至少补出 `8` 条新的可见视频，减少此前 `2-3` 条一批的碎片化加载。
- `HOME_FOLLOW` 单次动态请求目标提高，底部分页触发更稳定，也减少了偶发“拉到底无法继续加载”的情况。
- 过滤后的关注结果增加随机排序种子，避免每次都落成完全相同的显示顺序。

## 返回一致性
- 首页顶部标签页中的 `刷新 / 加载更多 / 点击视频 / 长按预览 / 直播点击` 现在都会先同步当前交互页。
- 从任意顶部标签进入视频详情后返回，会优先回到观看前所在的标签页，不再错误落回 `关注`。
