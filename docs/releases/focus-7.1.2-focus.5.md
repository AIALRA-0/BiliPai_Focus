# BliPai Focus v7.1.2-focus.5

发布日期：2026-03-24

## 更新摘要
- 本次为首页顶部标签返回一致性、首页“关注”极速刷新与随机性、以及首次使用声明入口补齐的稳定性版本。

## 首页顶部标签修复
- 首页现在会区分 `FocusSettings` 是否已真正完成加载；在设置尚未水合前，不再把当前顶部标签误判为“不可见”。
- 仅开启 `关注 / 热门 / 直播` 等部分标签时，从 `热门` 或 `直播` 进入视频页返回后，会继续停留在进入前的顶部标签，而不是错误掉回 `关注`。

## 首页“关注”提速
- 开启 Focus 关注过滤后，首页“关注”主路径改成按可见 UP 并发抓取用户动态，单轮最多并发 `8` 路。
- 一旦拿到首批 `8` 条新的可见视频就优先落 UI；刷新时不会先清空已显示内容，剩余内容继续后台补齐。
- 关注对象列表同步也改成首轮拿总数、后续并发分页，减少高关注量场景下的前置等待。

## 刷新随机性与首次使用声明
- 首页“关注”顺序改成“跨 UP 交错 + 每次刷新更换随机种子”，前几屏顺序更随机，同时保持去重稳定。
- 启动时的“首次使用声明”现在同时展示 `官方 GitHub / Focus GitHub / Telegram 交流群 / Telegram 频道` 四个入口；引导页原有 `Telegram 群` 入口继续保留。
