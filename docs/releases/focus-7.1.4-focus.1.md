# BliPai Focus v7.1.4-focus.1

发布日期：2026-03-24

## 更新摘要
- 本次同步上游 `BiliPai v7.1.4`，并继续收口首页“关注”的手动下拉刷新时序、新内容优先级，以及分页触发门闩。

## 上游 `7.1.4` 同步
- 合并官方 `v7.1.4` 的竖屏视频封面比例、封面到首帧衔接，以及竖屏进度拖动回弹修复。
- Focus 版继续保留独立包名、与官方版共存安装，以及只接收 Focus 仓库正式 `release` APK 的应用内更新通道。

## 手动刷新改成“结束后再切新列表”
- 首页“关注”手动下拉刷新时，当前会先保持已显示列表并进入刷新态。
- 新的关注视频排序结果会等刷新完成后再一次性切入，不再边下拉边刷新列表内容。

## 新内容优先靠前
- 刷新时新抓到的关注视频现在会优先排在旧内容前面，同时继续保持跨 UP 交错随机。
- 这样既不会每次都固定成同一批 UP 顶在前面，也更容易在刷新后先看到刚更新的内容。

## 严格 `16` 条一组与分页门闩
- 首页“关注”首次进入、刷新完成后，以及每次继续下滑，都继续严格按 `16` 条一组展示。
- FOLLOW 现在只有在“自动补批已解锁 + 观察到真实下滚 + 当前确实到达尾部”三者同时满足时，才会继续触发下一批，减少首屏偷偷灌几十条和到底偶发不继续加载的问题。
