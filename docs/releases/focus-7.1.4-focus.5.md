# BliPai Focus v7.1.4-focus.5

发布日期：2026-03-24

## 更新摘要
- 这一版只修正首页 FOLLOW 的排序刷新语义，重点把 `随机排序` 和 `UP聚类/时间排序` 的行为彻底区分开。

## 主要改动
- `随机排序` 刷新时继续保留换 seed、跨 UP 交错重排和新内容优先前置。
- `UP聚类倒序/正序` 与 `时间倒序/正序` 刷新时不再额外打乱，也不会再因为“新抓到的视频键”被强行插到前面。
- 非随机模式现在只会把新内容纳入当前可见池，然后严格按各自排序规则重新计算结果，排序更稳定、更符合预期。
