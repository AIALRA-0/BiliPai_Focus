# BliPai Focus v7.5.0-focus.1

发布日期：2026-04-06

## 更新摘要
- 这一版把 Focus 维护线直接同步到上游 `7.5.0`，完整吸收 `7.3.1 -> 7.5.0` 期间已经发布的空间页、播放器、直播、弹幕、动态页、缓存清理和构建验证改动，同时保留 Focus 的共存包名、Focus-only 更新源、FOLLOW 定制与首用声明入口。

## 主要改动
- 并入上游空间页聚合首屏架构：空间页会优先用聚合种子装配基础资料、默认 tab 和投稿上下文，再继续补齐投稿第一页和其他非首屏关键数据，减少首屏多请求拼装造成的抖动和误空态。
- 保留 Focus 的空间兼容层：搜索进入 `UP` 空间时，`SearchUpItem` 和空间投稿列表的关键数值字段继续兼容字符串数字；若命中“默认视频页 + 用户信息正常 + 首屏可疑空结果”，仍会刷新一次 `WBI key` 并执行单次重试。
- 并入上游 `7.3.2 -> 7.4.3` 的播放器和播放链稳定化，包括画质模式、封面与首帧过渡、播放诊断、seek 恢复、缓存清理选择器、直播前后台恢复、弹幕同步/过滤/行高和合集排序等能力。
- 并入上游 `7.5.0` 的动态页回到顶部、首页作者行布局优化与竖屏全屏切换过渡改进。
- 继续收口竖屏长宽比问题：从竖屏全屏内跳、外跳或切后台回来时，会先复位 inline 布局折叠偏移、视口变换和临时宽高覆盖，再恢复 shared player 和播放同步，尽量减少部分系统上的挤扁、黑边和错误比例首帧。
- Focus 版继续保持共存包名、Focus-only 更新源、首页 FOLLOW 分组/排序/自动同步、release-only 更新资产选择和首用声明里的 Focus GitHub / Telegram 入口。
