# BliPai Focus v7.9.6-focus.1

## 版本信息
- 基于上游 `BiliPai v7.9.6`
- Focus 版本：`7.9.6-focus.1 / 164`

## 本次同步
- 完整并入上游 `7.5.2 -> 7.9.6` 的正式更新，包括播放器、直播、弹幕、评论、搜索、空间、下载、设置和首页导航的主线修复与体验改进。
- 保留 Focus 共存包名、Focus-only 更新源、首页 FOLLOW 分组/排序/自动同步与首用声明入口。

## Focus 修复
- 修复动态默认多关注者页过滤后只显示几条动态、到底后不继续加载的问题；现在默认页会继续主动补页到更稳定的可滚动阈值，再交给底部 load more 接力。
- 修复 `7.5.x` 之后应用内“检查更新”容易失败的问题；更新检查现在会优先使用当前构建注入的 Focus 仓库坐标，并在异常时自动回退到 Focus 正式发布仓库。

## 发布说明
- 仅发布正式 `release` APK
- 与官方版继续保持可共存安装
