# BliPai Focus v8.0.0-Alpha3-focus.1

## 版本信息
- 基于上游 `BiliPai v8.0.0-Alpha3`
- Focus 版本：`8.0.0-Alpha3-focus.1 / 165`

## 本次同步
- 完整并入上游 `7.9.7 / 7.9.8 / 7.9.9 / 8.0.0-Alpha1 / Alpha2 / Alpha3` 的主线更新，包括搜索发现、列表观看进度、直播横屏聊天、评论保存图片二维码、图片预览文字显隐、合集订阅真实接口、下载重试与播放器 seek / 手势优化。
- 保留 Focus 共存包名、Focus-only 更新源、首页 FOLLOW 分组/排序/自动同步与首用声明入口。

## Focus 修复
- 修复搜索页进入或输入时的脆弱点：自动聚焦改成一次性保守请求，搜索建议映射也增加容错，降低搜索页进入和输入闪退概率。
- 修复拖动进度条后 UI 进度偶发冻结的问题：seek 会话现在具备超时脱困，避免底层播放位置迟迟不回流时把进度条永久卡住。
- 修复应用内更新对 `8.0.0-Alpha3-focus.1` 这类预发布 Focus 版本的比较问题，Alpha / RC / focus 组合版本现在能正确比较高低。
- 完成 Focus 现有分页稳定性补丁：搜索发现独立开关、共享分页策略，以及动态默认多关注者页的 `sourceHasMore / visibleHasMore / continuationAllowed` 状态拆分。

## 发布说明
- 仅发布正式 `release` APK
- 与官方版继续保持可共存安装
